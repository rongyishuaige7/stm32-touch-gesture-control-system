# 安全、网络与隐私边界

## 适用范围

这是桌面教学原型，不是生产级家电控制器、安防系统或安全控制器，禁止连接市电或安全关键负载。

## 网络边界

ESP-01S 建立公开配置的本地教学热点并提供无认证 HTTP：

- 不要把热点或 HTTP 服务桥接到公网、校园网或不可信局域网；
- 示例密码不应复用为个人、家庭、学校或企业 Wi-Fi 密码；
- HTTP 无 TLS、身份认证、会话、请求完整性或设备身份；
- `Access-Control-Allow-Origin: *` 仅用于隔离环境演示，不是生产安全设计；
- 诊断接口可能泄露设备运行细节，不应暴露给不可信用户。

## 硬件边界

- ESP-01S 只能使用匹配的 3.3 V 电源和逻辑电平；
- GPIO 只能驱动符合额定值的教学负载；
- 扩展继电器、灯带或电机前必须补充驱动、隔离和保护；
- 不要将本原型用于门锁、报警、医疗、工业或其他安全关键控制。

## 报告问题

如果仓库启用了 GitHub 私密漏洞报告，请优先使用该入口。否则只提交经过脱敏的最小问题描述，不要附带 Wi-Fi、令牌、账号、个人路径、设备标识或私网资产信息。

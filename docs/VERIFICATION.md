# 验证说明

> 状态日期：2026-07-17

## 一键本地门禁

```bash
bash scripts/verify.sh
```

门禁会在临时副本中执行：

1. 敏感信息、私钥、个人路径和不应公开配置检查；
2. 必需文件、生成物、大文件、BOM/SVG 与状态声明检查；
3. Python 源码契约测试；
4. PlatformIO 固件干净构建；
5. 核对候选目录没有跟踪构建产物。

门禁不烧录硬件，不连接 SoftAP，也不向 ESP-01S 发送真实 AT 指令。

## 已验证环境

```text
PlatformIO Core: 6.1.19
Platform: ST STM32 19.5.0
Board target: genericSTM32F103C8
Framework: STM32Cube
STM32CubeF1 package: 1.8.6
GNU Arm toolchain package: 1.70201.0
```

2026-07-17 的隔离构建结果：

```text
RAM:   4576 / 20480 bytes（22.3%）
Flash: 20228 / 65536 bytes（30.9%）
```

这些数字是构建证据，不证明实际板卡 Flash 容量、运行时栈余量、烧录成功或硬件行为。
